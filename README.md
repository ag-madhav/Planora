🏠 Planora — AI Floor Plan Generator

Planora is a web-based platform that generates personalized home layouts based on user preferences like house type, style, and number of bedrooms.

## 🚀 Live Demo

https://planora-9084f.web.app/

## ✨ Features

* AI-generated floor plan images using Pollinations AI
* User authentication with Firebase
* Interactive onboarding for design preferences
* Multi-floor support for villas and duplex homes
* Clean and responsive UI

## 🛠 Tech Stack

* React (Frontend)
* Firebase (Auth & Hosting)
* Pollinations AI (Image Generation)
* Express (Backend - optional)

## 📸 How it works

1. User selects house type, style, bedrooms
2. Input is converted into an AI prompt
3. Pollinations generates a floor plan image
4. User can view and save the design

## ⚙️ Setup Instructions

```bash
git clone https://github.com/ag-madhav/Planora.git
cd Planora
npm install
npm run dev
```

## 🔒 Environment Variables

Create a `.env` file:

```
GEMINI_API_KEY=your_key_here
```
